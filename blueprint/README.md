# Blueprint

LaTeX blueprint tooling (e.g. leanblueprint) for theorem-plan and proof-linked documentation. The project's hand-maintained blueprint-style docs (claim ID to target declaration) live in [docs/blueprints/](../docs/blueprints/); canonical mapping is in `corpus/papers/<paper_id>/mapping.json`. See [Contributor playbook – Blueprint–mapping contract](../docs/contributor-playbook.md#blueprintmapping-contract) (under schema versioning).
